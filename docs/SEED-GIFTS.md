# BeTalent Gift Catalog Seed

## Overview

The gift seed populates the `Gift` table with 50+ premium, music/performance-themed gifts. It is the single source of truth for the BeTalent fan-support catalog.

## Gift Fields

| Field | Description |
|-------|-------------|
| `id` | Auto-generated by Prisma (cuid) |
| `slug` | Unique identifier (e.g. `music-note`, `platinum-record`) |
| `name` | Display name |
| `icon` | Icon key for GiftIcon component mapping |
| `animationType` | Animation type for GiftCelebration |
| `coinCost` | Cost in coins |
| `rarityTier` | BASIC \| PREMIUM \| RARE \| LEGENDARY |
| `isActive` | Whether the gift is available |

## Source of Truth

- **Catalog:** `src/constants/giftCatalog.ts`
- **Seed script:** `scripts/seed-gifts.ts`
- **Icon mapping:** `src/components/gift/GiftIcon.tsx`
- **Animation config:** `src/constants/giftAnimationRegistry.ts`

## How to Run

**From project root:**

```bash
npm run db:seed-gifts
```

**Alternative (direct tsx):**

```bash
npx tsx scripts/seed-gifts.ts
```

**Prerequisites:**
- Database migrated (`npx prisma db push` or `npx prisma migrate dev`)
- `.env` with `DATABASE_URL` set

## Behavior

- **Safe to run multiple times:** Uses upsert by `slug`
- **Adds new gifts** from the catalog
- **Updates existing gifts** if catalog changed
- **Does not delete** any gifts
- **Requires 50+ gifts** in catalog (script validates)

## Gift Categories

| Tier | Coin range | Examples |
|------|------------|----------|
| BASIC | 5–25 | Music Note, Clap, Microphone |
| PREMIUM | 50–150 | Headphones, Drum Beat, Golden Mic |
| RARE | 200–600 | Piano, Golden Score, Diamond |
| LEGENDARY | 750–2000 | Platinum Record, Superstar, BeTalent Crown |
